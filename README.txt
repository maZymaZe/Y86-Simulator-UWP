目标 Universal Windows
目标版本 Windows 10，version 1903（10.0；版本18362）
最低版本 Windows 10，version 1903（10.0；版本17763）

安装方法说明
0，确认环境。
1，运行r1_1.0.2.0_Test\r1_1.0.2.0_x86.cer安装证书
2，确认开发者设置中进入开发者模式，“允许从任意源安装程序”。
3，用powershell执行r1_1.0.2.0_Test\Install.ps1。
4，确认安装。

使用方法说明：
openfile导入.yo文件
reset可以清理已导入文件
refresh可以恢复文件的初始状态
previous可以倒退1步
play/pause控制代码执行
next可以单步
speedslider可以控制play的执行速度
